# SISWIT Employee Attendance Bot

A Telegram bot for automating daily work/attendance updates, with Excel and Google Sheets persistence, and admin reporting/approval workflows.

## Features

- Daily attendance submission via Telegram group
- Excel and Google Sheets dual persistence
- Re-submission, edit, and leave approval workflows
- Admin reports (daily, weekly, monthly)
- Role-based access (Owner, HR, Employee)
